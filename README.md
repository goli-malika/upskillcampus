# 🔐 Secure Password Manager Using Python

A secure and modern Password Manager web application developed using **Python**, **Flask**, **SQLite**, **HTML**, **CSS**, and **Bootstrap**. The application helps users securely store, manage, and retrieve passwords for different websites while protecting sensitive information through encryption.

---

## 📌 Project Overview

Managing multiple passwords can be difficult and insecure. This project provides a secure solution where users can register, log in, generate strong passwords, and safely manage their credentials.

This project was developed as part of the **Upskill Campus Summer Internship 2026** in collaboration with **The IoT Academy** and **UniConverge Technologies Pvt. Ltd.**

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/upskillcampus.git
```

### Navigate to the project

```bash
cd upskillcampus
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the environment

Windows

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
